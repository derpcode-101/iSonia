Thinkpad

My friend gifted me an empty ThinkPad to play around with. I wanted a computer with Linux installed, so I decided to set up Linux on there. The ThinkPad did not have an internal hard-drive, so here is what I did to install it:

Equipment:
- SSD Hard drive
- SATA to USB cable

Plan:
- Download Linux ubuntu from personal computer onto new SSD.

Downloaded Ubuntu from here.

Step-by-step process:

First, confirm that what you downloaded is the Ubuntu ISO file (it should end in .iso)

Ensure that the SSD is formatted properly:

  1. Open Disk Utility on your Mac
  2. Select the SSD from the left sidebar
  3. Click "Erase"
  4. Choose "MS-DOS (FAT)" or "ExFAT" as the format
  5. Choose "GUID Partition Map" as the scheme
  6. Give it a name and click "Erase"
To create the bootable drive, you'll need to use Terminal. The built-in 'dd' command can do this:

  1. Open Terminal
  2. Type diskutil list to identify your SSD's identifier (it will be something like /dev/disk2)
  3. Unmount the SSD with: diskutil unmountDisk /dev/diskN (replace N with your disk number)
  4. Use dd to write the ISO: sudo dd if=/path/to/ubuntu.iso of=/dev/rdiskN bs=1m
  5. (replace /path/to/ubuntu.iso with the actual path to your ISO file, and N with your disk number)
  6. Enter your password when prompted
  7. Wait for the process to complete (it might take a while and won't show progress)
Once completed, safely eject the SSD using the command: diskutil eject /dev/disk4
